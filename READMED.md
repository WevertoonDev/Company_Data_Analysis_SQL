🧠 Anotações SQL - LIMIT

A cláusula LIMIT é usada para restringir o número de linhas retornadas por uma consulta.

Exemplo:
SELECT * FROM Colaboradores
LIMIT 5;
Caso de uso:

Útil ao trabalhar com grandes volumes de dados para visualizar uma amostra ou obter os primeiros resultados.

Aprendizado:
Ajuda a melhorar o desempenho
Comumente usada com ORDER BY
